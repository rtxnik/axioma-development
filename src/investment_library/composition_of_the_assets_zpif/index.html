  • Состав имущества ЗПИФ

Учредители доверительного управления совместно с управляющей компанией могут определить состав имущества, подлежащего внесению в оплату инвестиционных паев фонда, исходя из целей и задач, решаемых с помощью паевого инвестиционного фонда.

В состав имущества, включаемого в комбинированный закрытый паевой инвестиционный фонд (в соответствии с инвестиционной декларацией фонда), могут входить:

  • доли в уставных капиталах предприятий;
  • недвижимость и имущественные права на объекты недвижимости;
  • земельные участки любого назначения;
  • имущественные права по договорам участия в долевом строительстве согласно Федеральному закону №214-ФЗ;
  • имущественные права на объекты недвижимости после завершения их строительства;
  • имущественные права по договорам, на основании которых осуществляется реконструкция объектов недвижимости;
  • имущественные права по обязательствам из кредитных договоров и договоров займа;
  • ценные бумаги, включая акции, облигации и государственные ценные бумаги;
  • производные финансовые инструменты;
  • права требования по денежным обязательствам и договорам займа;
  • права требования по договорам, заключённым для целей доверительного управления;
  • права застройщика и аренды;
  • права участия в уставных капиталах иностранных коммерческих организаций;
  • проектная документация для строительства и реконструкции объектов недвижимости;
  • иные активы, утверждённые правилами доверительного управления.

Имущество, составляющее паевой инвестиционный фонд, подлежит обособлению и учитывается управляющей компанией на отдельном балансе с ведением самостоятельного учёта.
